# OpenClaw Assistant

Desktop app for managing your WhatsApp AI assistant. No technical knowledge required.

## Download

Go to [Releases](https://github.com/kktvg001-dot/xbb-desktop/releases) and download:
- **Windows:** `OpenClaw-Assistant-Setup-x.x.x.exe`
- **Mac:** `OpenClaw-Assistant-x.x.x.dmg`
- **Linux:** `OpenClaw-Assistant-x.x.x.AppImage`

## What it does

1. **Installs everything** — Claude Code + OpenClaw, one click
2. **Chat interface** — describe problems in plain English, AI fixes them
3. **Status dashboard** — see if WhatsApp is connected
4. **Settings** — configure API keys and proxy

## For developers

```bash
git clone https://github.com/kktvg001-dot/xbb-desktop.git
cd xbb-desktop
npm install
npm start
```

## Building installers

```bash
npm run package:win    # Windows .exe
npm run package:mac    # Mac .dmg
npm run package:linux  # Linux .AppImage
```
